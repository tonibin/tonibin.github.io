Introduction

Welcome to soft, the Berkeley Snobol4 interpreter. soft is simple but elegant, full of features and with a full documentation. It's a complete and easy tool, useful for learning this wonderful language for the humanities. It includes the full Berkely Snobol4 system, with some update from the Bell Snobol4, trying to be the Unix best bridge between the two systems.

Installation from sources

While the Windows version executable can be downloaded directly from the link you find at the bottom of this page, the soft executable for the Linux and Unix environments, must be compiled and installed before usage. I suggest the following very easy procedure in 4 steps (use the terminal):

1) Copy the package somewhere, and unpack it; it will unpack into a directory named soft.XXX.src (where 'XXX' stands for the version string);

2) Get into soft source directory, open the file custom.h and change what you think must be changed. The file has auto-explicative comments;

3) Now type the following (be sure to have gcc installed):

$ make
$ make install

The latter may require root privileges if you try to install soft into system directories, as suggested by default in the Makefile. In this case, if you have the administrative access, as a common user, type

$ sudo make install

and the program will install.

4) now type the following to test the correct installation:

$ soft --version

to see the version number, or better

$ soft --help

to see all basic options of soft. Here it is! soft is installed and can be run! Remember also to read the manual, to know the internal secrets of soft!
