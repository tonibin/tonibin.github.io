Introduction

Welcome to decb, the DEC-20 BASIC interpreter. decb is based on the DEC manuals, and follows the original syntax. It comes with a full documentation. This program was the core of tbas, my current. historical unrelated, BASIC interpreter

Installation from sources

decb 's executable for the Linux and Unix environments must be compiled and installed before usage. I suggest the following very easy procedure in 4 steps (use the terminal):

1) Copy the package somewhere, and unpack it; it will unpack into a directory named decb.XXX.src (where 'XXX' stands for the version string);

2) Get into decb source directory, open the file custom.h and change what you think must be changed. The file has auto-explicative comments;

3) Now type the following (be sure to have gcc installed):

$ make
$ make install

The latter may require root privileges if you try to install decb into system directories, as suggested by default in the Makefile. In this case, if you have the administrative access, as a common user, type

$ sudo make install

and the program will install.

4) now type the following to test the correct installation:

$ decb --version

to see the version number, or better

$ decb --help

to see all basic options of decb. Here it is! decb is installed and can be run! Remember also to read the manual, to know the internal secrets of decb!
