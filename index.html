Hobbies the Hobbiest

Welcome to HtH my personal Web Page. I'm Antonio Maschio, from Italy. I like simple pages, so this is a simple page. I'm specialized (let me show off a bit) in ancient programming languages, for which I create interpreters in C. It's my fun, it's my joy, and if this can serve someone for historical reasons, it's my happiness too!

The Free Software here linked is GPL and freely downloadable. If you don't find the COPYING text file in some package, write me and I'll provide one and integrate the package.
I hope you'll find my software as interesting as I do.

Here are listed my console software projects (developed with gcc); they all must be used through a terminal (the dear old way, no windows, no mice):

Weird Languages
  • owl - Obfuscated Weird Language - A character-based language inspired by 'false'. Designed for UNIX and Windows.

  • cdd - scripts collection in bash for maintaining and querying a textual musical compact-discs database - Designed for UNIX.
Historical Languages
  • taxi - an Algol60 interpreter. Designed for UNIX and Windows.

  • apple - a powerful APL interpreter - Designed for UNIX.

  • tbas - a powerful BASIC interpreter - Designed for UNIX and Windows.

  • tetra - a Forth-79 interpreter - Designed for UNIX.

  • dib - the Dartmouth BASIC interpreter for version III (1966) - Designed for UNIX.

  • decb - the DEC-20 BASIC interpreter (1974) - Designed for UNIX.

  • soft a Snobol4 interpreter - Designed for UNIX. (still to be loaded)
